How To Use

  • The three node manipulation methods on the right will modify a tree data structure that will be animated on the canvas on the lefthand side according to the specifics of the data structure.
  • For additions, you can add mulitple nodes into a tree by inserting each key item that you want to add with commas in between. For example we can add 3 separate nodes with the command 3,4,5.

Previous Tree

  • This will animate the most recent tree modified by the user in a modal popup.

Insert History

  • This will give you the history of nodes that you have inserted so that you can share or reproduce later on. It is formatted like a multi-node addition operation which you will put into the add node operation
